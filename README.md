## Installation

Just add it to your ``rebar.config`` deps:

```erlang
    {deps, [
        ...
        {ctxsol, ".*", {git, "git@github.com:YOURNAME/ctxsol.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
